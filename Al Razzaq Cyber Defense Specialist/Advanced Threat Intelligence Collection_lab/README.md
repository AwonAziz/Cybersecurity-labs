This lab focuses on building a complete threat intelligence pipeline using Python. Students will collect threat indicators from multiple sources, validate and normalize the data, calculate risk scores, and generate automated threat intelligence reports in multiple formats.

Objectives

By the end of this lab, students will be able to:

Collect threat intelligence data from multiple sources
Normalize and standardize threat indicators
Validate IP addresses, domains, and file hashes
Calculate risk scores for threat indicators
Generate automated threat intelligence reports
Build a complete threat intelligence pipeline

Tasks Covered
Environment setup and configuration
Threat intelligence data collection
Indicator validation and enrichment
Data normalization and deduplication
Risk scoring and severity classification
Threat intelligence report generation
Automated pipeline execution
Expected Outcomes

After completing this lab, you should have:

Threat intelligence collector script
Data normalization script
Report generation script
Automated pipeline script
JSON, HTML, and text reports
Complete threat intelligence workflow
